
# Restaurant Management System (C)

## Overview

A console-based **Restaurant Management System** developed in **C** using file handling. The system supports **admin product management** and **customer registration and ordering**, with persistent data storage.

---

## Features

### Admin

* Secure login with limited attempts
* Product registration
* View available products

### Customer

* New customer registration
* Existing customer login
* View product list
* Place orders and view total cost

---

## Technologies

* **Language:** C
* **Core Concepts:** Structures, File Handling, String Handling, Authentication Logic

---

## Data Files

| File             | Description       |
| ---------------- | ----------------- |
| `restaurant.txt` | Admin credentials |
| `rest.txt`       | Customer records  |
| `product.txt`    | Product details   |

---

## How to Run

```bash
gcc restaurant.c -o restaurant
./restaurant
```
